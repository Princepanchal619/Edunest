# How to Seed Dummy Data

After your backend server is running, open your browser and go to:

```
http://localhost:4000/api/v1/seed
```

This will automatically create:
- ✅ 4 Categories (Web Dev, Data Science, Mobile Dev, UI/UX)
- ✅ 5 Published Courses with sections and lessons
- ✅ 1 Demo Instructor account

**Demo Instructor Login:**
- Email: `instructor@edunest.com`
- Password: `Demo@1234`

You can run it multiple times safely — it skips already existing records.
